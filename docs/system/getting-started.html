Getting Started

Quick start guide for Beacon Trading System

Run the System (Easy Mode)

# From the project root:
python3 beacon-run.py -i config/system/startBeacon.json

That's it! The system will:

  1. Start matching engine
  2. Start your algorithm
  3. Generate market data
  4. Show trading statistics

Directory Organization

beacon/
├── beacon-run.py                 # Main entry point
├── beacon-build.py               # Build system
├── beacon-config.py              # Configuration management
├── config/                       # Configuration files
│   ├── system/                   # System configurations
│   ├── algorithm/                # Algorithm settings
│   └── matching_engine/          # Engine settings
├── src/apps/                     # Application code
│   ├── generator/                # Market data generation
│   ├── matching_engine/          # Order matching
│   ├── client_algorithm/         # Trading algorithms
│   ├── playback/                 # Historical replay
│   └── orchestrator/             # System coordination
├── include/                      # Header files
├── docs/                         # Documentation
└── outputs/                      # Generated reports

First Time Setup

# 1. Build the system
python3 beacon-build.py --release

# 2. Run with default configuration
python3 beacon-run.py -i config/system/startBeacon.json

Daily Workflow

# Configure your trading session
python3 beacon-config.py

# Run the trading system
python3 beacon-run.py -i config/system/startBeacon.json

Troubleshooting

# Check system status
python3 beacon-run.py --status

# Clean rebuild
python3 beacon-build.py --clean

# View logs
tail -f outputs/trade_report.log
