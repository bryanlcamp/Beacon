üì° Pulse

Network Simulation & Data Broadcasting

Controls data flow and simulates real-world network conditions. The chaos engineering platform that tests strategies against infrastructure disasters.

Network Reality Simulation
  • UDP multicast for market data distribution
  • Configurable latency spikes and packet ordering
  • Per-product or per-dataset network control
  • Exchange outage and disaster simulation
Chaos Engineering
  • Exchange dark periods
  • Malformed/broken message injection
  • Message flood stress testing
  • Administrative message simulation
Data Flow Architecture
1. Pulse reads Author-generated datasets into memory
2. UDP multicast streams data to Match and Core simultaneously
3. Match posts liquidity immediately (timing advantage)
4. Core processes data + sends TCP orders (realistic delay)
5. Match executes trades and sends back to Pulse
6. Pulse injects trades into live data stream
