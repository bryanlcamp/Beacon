# Beacon Logo Assets

Professional branding assets for the Beacon Trading System.

## Logo Variations

### `beacon-logo.svg` - Standard Logo
- **Size:** 120x40px  
- **Usage:** Navigation bars, headers, general branding  
- **Background:** Professional blue gradient  
- **Format:** Scalable SVG  

### `beacon-icon.svg` - Icon Only  
- **Size:** 60x60px  
- **Usage:** Favicons, app icons, compact spaces  
- **Background:** Circular blue gradient  
- **Format:** Scalable SVG  

### `beacon-header.svg` - Header Logo
- **Size:** 200x60px  
- **Usage:** Page headers, documentation banners  
- **Background:** Extended blue gradient with tagline  
- **Format:** Scalable SVG  

## Usage Guidelines

### Colors
- **Primary Gray:** #1a1a1a (dark charcoal)
- **Secondary Gray:** #2d2d2d (medium gray)  
- **Accent Gray:** #888888 (light gray)
- **Text:** #ffffff (white)

### Implementation

**HTML:**
```html
<img src="docs/assets/beacon-logo.svg" alt="Beacon Trading System" height="40">
```

**Markdown:**
```markdown
![Beacon Trading System](docs/assets/beacon-logo.svg)
```

**CSS Background:**
```css
.beacon-logo {
    background-image: url('assets/beacon-logo.svg');
    background-size: contain;
    background-repeat: no-repeat;
}
```

## Professional Standards

- **Consistent Usage:** Always use official logo files
- **Clear Space:** Maintain minimum 10px clearance around logo
- **Scaling:** SVG format ensures crisp rendering at all sizes
- **Contrast:** White text ensures readability on any background
