Client Algorithm

Professional trading algorithm development framework

Overview

The Client Algorithm framework provides a professional environment for developing and testing trading strategies. It includes built-in algorithms like TWAP (Time-Weighted Average Price) and supports custom algorithm development with comprehensive market data processing and order management capabilities.

Key Features

  • Built-in TWAP algorithm implementation
  • Custom algorithm development framework
  • Real-time market data processing
  • Protocol-aware order entry (OUCH, Pillar, iLink)
  • Risk management and position tracking

Usage

Algorithms connect to market data feeds via UDP and submit orders to the matching engine via TCP. The framework handles protocol messaging, connection management, and provides callback interfaces for market data and execution events.

Configuration

Algorithm behavior is controlled through JSON configuration files that specify trading parameters, risk limits, symbol lists, and execution strategies. Each algorithm can be independently configured and deployed.
