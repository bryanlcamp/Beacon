Market Data Generator

Realistic market data feed generation for trading algorithm testing

Overview

The Market Data Generator creates realistic, high-frequency market data feeds that simulate real exchange behavior. It generates tick-by-tick market data including prices, volumes, and order book updates for testing trading algorithms under realistic market conditions.

Key Features

  • Multi-symbol market data generation (AAPL, MSFT, GOOGL, etc.)
  • Configurable price ranges, volatility, and volume patterns
  • Protocol-aware output (NASDAQ ITCH, NYSE Pillar, CME MDP)
  • Real-time UDP broadcast for low-latency consumption
  • Burst mode support for high-throughput testing

Usage

The generator is typically launched as part of a complete trading system via beacon-run.py, but can also be run independently for data generation testing.

Configuration

Generator behavior is controlled through JSON configuration files that specify symbol parameters, price ranges, message rates, and output protocols. The system supports both pre-configured templates and custom configurations.
