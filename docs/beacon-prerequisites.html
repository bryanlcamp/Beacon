Prerequisites

System requirements and dependencies for Beacon.

Quick Start: Install Git → Clone repo → Run verification script
git clone https://github.com/bryanlcamp/beacon.git && cd beacon && ./scripts/check-prerequisites.sh

Beacon includes an automated verification script that checks all system requirements. If the script passes, you're ready to build and run Beacon. If it fails, it will tell you exactly what's missing and you can refer to the requirements below for installation instructions.

Requirements

Component Requirement Check
OS Linux (Ubuntu 20.04+) or macOS (10.15+) uname -a
CMake 3.16+ cmake --version
GCC/Clang GCC 9+ or Clang 10+ gcc --version
Python 3.6+ python3 --version
Git 2.20+ git --version

Platform-Specific Installation

Ubuntu/Debian

# Update package manager
sudo apt update

# Install build tools
sudo apt install -y build-essential cmake git python3

# Install additional tools  
sudo apt install -y python3-pip

# Verify installation
cmake --version && gcc --version && python3 --version && git --version

macOS

# Install Xcode Command Line Tools
xcode-select --install

# Install Homebrew (if not already installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install dependencies
brew install cmake git python3

# Verify installation
cmake --version && clang --version && python3 --version && git --version

Quick Check

Run the automated prerequisites checker:

./scripts/check-prerequisites.sh
