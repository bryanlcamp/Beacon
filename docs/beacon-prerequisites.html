Prerequisites

System requirements and dependencies for Beacon.

System Requirements

Component Requirement Check
OS Linux (Ubuntu 20.04+) or macOS (10.15+) uname -a
CPU x86_64 or ARM64, multi-core recommended lscpu | grep Architecture
Memory 8 GB minimum, 16+ GB recommended free -h
Storage 10 GB free space, SSD recommended df -h

Required Dependencies

Component Version Check
CMake 3.16+ cmake --version
GCC/Clang GCC 9+ or Clang 10+ gcc --version
Make/Ninja Any recent version make --version
Git 2.20+ git --version
Note: pthread, sockets, nlohmann/json, and GoogleTest are included or built-in.

Platform-Specific Installation

Ubuntu/Debian

# Update package manager
sudo apt update

# Install build tools
sudo apt install -y build-essential cmake git

# Install additional tools
sudo apt install -y ninja-build pkg-config

# Verify installation
cmake --version && gcc --version && git --version

macOS

# Install Xcode Command Line Tools
xcode-select --install

# Install Homebrew (if not already installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install dependencies
brew install cmake git ninja

# Verify installation
cmake --version && clang --version && git --version

CentOS/RHEL/Fedora

# CentOS/RHEL
sudo yum groupinstall "Development Tools"
sudo yum install cmake git

# Fedora
sudo dnf groupinstall "Development Tools"
sudo dnf install cmake git ninja-build

# Verify installation
cmake --version && gcc --version && git --version

Network Setup

⚠️ Firewall: Allow ports 8000-9000 (TCP/UDP) for Beacon communication. 
# Ubuntu/Debian
sudo ufw allow 8000:9000/tcp && sudo ufw allow 8000:9000/udp

# CentOS/RHEL/Fedora
sudo firewall-cmd --permanent --add-port=8000-9000/tcp
sudo firewall-cmd --permanent --add-port=8000-9000/udp
sudo firewall-cmd --reload

# Test localhost connectivity
ping 127.0.0.1
