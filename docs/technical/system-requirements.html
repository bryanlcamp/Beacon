System Requirements

Beacon runs on all major operating systems and requires only a few modern tools.

Supported Operating Systems

  • Linux (Ubuntu 20.04+ recommended)
  • macOS (Monterey+)
  • Windows 10/11 (WSL2 recommended)

Software & Network

  • Python 3.9+
  • Node.js 18+
  • Modern web browser (Chrome, Firefox, Edge, Safari)
  • Stable internet connection
  • Ports 80, 443, 3000, 8000 open (as needed)
