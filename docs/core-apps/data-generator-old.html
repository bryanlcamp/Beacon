Market Data Generator

Professional Exchange Protocol Data Generation

🚀 Quick Start

Generate realistic market data in seconds with the modernized, simplified configuration format.

./build/src/apps/generator/generator -i config.json -o market_data.bin
./build/src/apps/generator/generator -i config.json -o data.csv --csv

⚙️ Simple Configuration Format

The generator now uses a simplified JSON format with just 5 parameters. Gone are the complex Wave/Burst configurations!

{ "exchange": "nasdaq", "message_count": 10000, "symbols": ["AAPL", "MSFT", "GOOGL"], "trade_probability": 0.15, "flush_interval": 1000 }
🎯 Parameter Guide:
  • exchange: nasdaq | cme | nyse
  • message_count: Total messages to generate
  • symbols: Array of ticker symbols (equal distribution)
  • trade_probability: 0.0-1.0 (probability of trades vs orders)
  • flush_interval: Messages before buffer flush

📋 Command Line Usage

# Generate binary ITCH data (production mode) ./generator -i config.json -o market_data.itch # Generate human-readable CSV for analysis ./generator -i config.json -o data.csv --csv # Short form with CSV output ./generator -i config.json -o data.csv -c

Required Parameters:

  • -i, --input <file> - Configuration file (JSON format)
  • -o, --output <file> - Output file path

Optional Parameters:

  • -c, --csv - Generate human-readable CSV output
  • -h, --help - Show help message

🔧 UI Integration

The generator works seamlessly with the Beacon UI system:

python3 beacon-run.py --dry-run -i config/system/startBeaconNasdaq.json
python3 beacon-run.py -i config/system/startBeaconNasdaq.json -d 60
The UI automatically converts legacy Wave/Burst configurations to the new simplified format, ensuring backward compatibility.
