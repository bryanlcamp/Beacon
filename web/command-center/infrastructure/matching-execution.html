Matching and Execution

Exchange-realistic order matching engine with institutional-grade execution logic.

Matching Engine Features

  • Price-Time Priority: Standard FIFO matching algorithm
  • Pro-Rata Allocation: Volume-based order allocation for large fills
  • Hidden Order Support: Iceberg and reserve order handling
  • Order Types: Market, limit, stop, and conditional orders
  • Cross Trading: Internal order crossing before external routing

Execution Logic

  • Realistic Fill Behavior: Market impact and slippage modeling
  • Partial Fills: Progressive execution for large orders
  • Reject Handling: Exchange-realistic order rejection scenarios
  • Timing Simulation: Accurate execution latency modeling
  • Risk Controls: Position limits and exposure management

Performance Characteristics

Market Simulation
