Web Server

Integrated Flask web server providing configuration interfaces and system monitoring capabilities.

Server Features

  • Configuration Interface: Web-based parameter configuration for all modules
  • Real-time Monitoring: Live system metrics and performance dashboards
  • API Endpoints: RESTful API for external system integration
  • File Management: Dataset and configuration file upload/download
  • Session Management: Persistent user sessions and preferences

Security and Access

  • Local Only: Server binds to localhost for security
  • No External Dependencies: Runs completely offline
  • File System Isolation: Restricted access to designated directories
  • Input Validation: All user inputs validated and sanitized
  • Session Security: Secure session token management

Performance Characteristics

  • Low Overhead: Minimal CPU and memory impact
  • Fast Response: Sub-millisecond API response times
  • Concurrent Users: Multiple browser sessions supported
  • Hot Reload: Configuration changes applied without restart

Configuration Management

  • JSON Storage: Human-readable configuration format
  • Version Control: Configuration history and rollback support
  • Template System: Predefined configuration templates
  • Export/Import: Configuration sharing and backup

Integration Capabilities

  • C++ Backend: Direct integration with core Beacon modules
  • IPC Communication: Inter-process communication for real-time data
  • File System Events: Automatic detection of external changes
  • Plugin Architecture: Extensible interface for custom modules

Usage

The web server automatically starts with Beacon applications and provides access through standard web browsers. Learn more about Build System integration.
