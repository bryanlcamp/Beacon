Quick Setup

Creating a complete pipeline takes just a few steps:

  • Configure Symbols: Select your trading instruments and parameters
  • Generate Dataset: Create market data with realistic patterns
  • Start Playback: Stream data using native exchange protocols
  • Connect Algorithm: Plug in your trading logic with low-latency interface
  • Monitor Results: Track performance metrics in real-time

Performance Benefits

The complete pipeline delivers:

  • 4.5μs median latency: From market data to order execution
  • Realistic simulation: True exchange behavior and timing
  • Scalable testing: From single instruments to full portfolios
  • Algorithm isolation: Test strategies without market risk
