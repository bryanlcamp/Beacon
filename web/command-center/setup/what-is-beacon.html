Beacon's Purpose.

Live Simulator for HFT Algorithms.
Easy to use. Plugin-in architecture. Refine. Explore.

System Overview.

Beacon is a live simulator for high-frequency trading algorithms. Full end-to-end system. Guided UIs for each component — simple by default, powerful when you dig deeper. Generate datasets. Send and receive market data via real exchange protocols. Replay at variable speeds. Inject bursts and events. Interactive FIFO matching engine. Test strategies offline with microsecond latency. Refine. Iterate.

System Usage.

Beacon models a complete trading pipeline: create market data, stream it like an exchange, process it with your algorithm, and execute against a matching engine.

Benefits

  • Minimal Setup. One interface. Plug in your algorithm. Simulate. Iterate.
  • Create Datasets. Create the market. Customize scenarios. Pinpoint weaknesses. Refine.
  • Stream Datasets. Replay at variable speed. Inject configurable events.
  • Matching Engine. FIFO matching. Simulate delays, rejects, and liquidity.
  • Exchange Ready. Real protocols. Market data. Algorithm. Execution.
  • Quality Infrastructure. 4μs tick-to-trade latency on a laptop.
  • Offline Mode. No internet connection required.
