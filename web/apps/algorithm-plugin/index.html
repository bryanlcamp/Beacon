Algorithm Plugin

Connect your trading algorithms with minimal latency interface for high-frequency strategy testing.

Algorithm Integration

  • Low-Latency Interface: 4.5μs market data feed with minimal processing overhead
  • Order Management: Submit, modify, and cancel orders with exchange-realistic timing
  • Position Tracking: Real-time position and P&L monitoring with accurate accounting
  • Risk Controls: Built-in position limits and risk management framework
  • Performance Analytics: Detailed execution metrics and latency measurement

Supported Interfaces

  • C++ Native API: Direct library integration for maximum performance
  • Python Bindings: High-level interface for rapid algorithm development
  • FIX Protocol: Industry-standard messaging for existing systems
  • REST API: HTTP interface for web-based algorithm frameworks
  • WebSocket Streaming: Real-time data streaming for web applications

Algorithm Lifecycle

  • Initialization: Algorithm setup and parameter configuration
  • Market Data Feed: Real-time or simulated market data streaming
  • Strategy Execution: Algorithm decision making and order generation
  • Order Processing: Order routing and execution confirmation
  • Performance Monitoring: Real-time analytics and risk monitoring

Integration

Algorithm Plugin receives market data from Market Data Playback and sends orders to Matching Engine for complete strategy testing.
