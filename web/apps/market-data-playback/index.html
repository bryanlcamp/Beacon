Market Data Playback

Stream generated or imported market data with precise timing control and native protocol support.

Playback Features

  • Precise Timing Control: Configurable playback speed from real-time to accelerated
  • Multiple Data Sources: Support for generated datasets and imported market data
  • Protocol Streaming: Native CME, NYSE, NASDAQ format output
  • Network Distribution: UDP multicast and TCP streaming options
  • Session Management: Market session simulation with proper timing

Performance Characteristics

  • Low Latency Streaming: Sub-microsecond message dispatch
  • High Throughput: 2M+ messages per second sustained rate
  • Memory Efficient: Streaming architecture with minimal memory footprint
  • Deterministic Timing: Consistent message timing under all conditions

Configuration Options

  • Data Source Selection: Choose from available datasets or import external data
  • Playback Speed: Real-time, accelerated, or custom timing control
  • Network Settings: Configure output protocols and network interfaces
  • Session Control: Start/stop/pause streaming with precise control

Integration

Market Data Playback works with Dataset Generator output files and streams to Algorithm Plugin modules for complete testing workflows.
