Matching Engine Features

Exchange-realistic order matching with institutional-grade logic:

  • Price-Time Priority: Standard FIFO matching algorithm with microsecond precision
  • Pro-Rata Allocation: Volume-based order allocation for large fills
  • Hidden Order Support: Iceberg and reserve order handling
  • Order Types: Market, limit, stop, and conditional orders
  • Cross Trading: Internal order crossing before external routing

Execution Logic

Realistic execution behavior that mirrors real exchange conditions:
