# Beacon Trading System

Professional C++ high-frequency trading platform with low-latency market data generation, algorithm development, and order execution simulation.

## Quick Start

```bash
python3 beacon-build.py --release
python3 beacon-run.py -i config/system/startBeacon.json
```

## Features

- Ultra-low latency market data processing
- Multi-exchange protocol support (NASDAQ OUCH, NYSE Pillar, CME iLink)
- Professional algorithm development framework
- Real-time order book and matching engine simulation
- Comprehensive performance monitoring
